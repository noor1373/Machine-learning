# Machine Learning Course

This repository contains my work and notes from a machine learning course focused on Scikit-learn and PyTorch.

The course builds on prior knowledge in:
- Data processing
- Data visualization
- Statistics
- Linear algebra

The first part of the course focuses on classical machine learning using Scikit-learn.  
The second part focuses on deep learning and neural networks using PyTorch.

This repo includes exercises, experiments, and implementations of different machine learning models.
